课程内容

HTML 和 CSS 基础知识：

  • HTML 元素标签的使用
  • HTML 文档基本结构
  • CSS 规则

HTML 元素标签的使用

元素标签的组成:

  1. 开始标签（The opening tag)
  2. 结束标签 (The closing tag)
  3. 内容 (The content)

元素标签有“内联元素”和“块级元素”，还有空元素。 元素也可以有属性(Attribute)，了解更多

HTML 文档基本结构

HTML 文档的基本结构很简单，包括声明、根元素(html、head、body)

HTML 各元素都是一个个矩形，设计一个网页，可以由大到小去分解每个元素，

CSS 规则

CSS 是用来为网页添加样式的。

  • 选择符(Selector)
  • 声明(Declaration)
  • 属性(Properties)
  • 属性值(Property Value)

选择符包括：元素选择符、ID选择符、class选择符、属性选择符、伪类选择符…………， 也可以自定义，详见 CSS 参考
